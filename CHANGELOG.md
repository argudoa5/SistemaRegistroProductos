# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

## [1.0.0] - 2026-05-31

### Added
- Creación de la API REST para la gestión de productos utilizando Python y FastAPI.
- Integración del programa principal `app.py` como archivo base de ejecución de la API.
- Implementación del endpoint `GET /productos` para listar los productos registrados.
- Implementación del endpoint `POST /productos` para registrar nuevos productos.
- Definición del modelo de datos `Producto` con los campos `nombre`, `precio` y `stock`.
- Generación automática de documentación mediante OpenAPI y Swagger UI.
- Creación del archivo `OpenAPI.yaml` con la especificación de la API.
- Creación del archivo `README.md` con información del proyecto, instalación y uso.
- Inclusión de la licencia MIT mediante el archivo `LICENSE`.
- Configuración del repositorio GitHub para el control de versiones.
- Integración de Jira para la planificación y seguimiento de tareas del proyecto.
- Uso de Confluence para la documentación técnica y funcional del sistema.

### Documentation
- Documentación de los endpoints de la API.
- Descripción de la estructura del proyecto.
- Documentación de requisitos previos e instalación.
- Inclusión de ejemplos de uso y pruebas mediante Swagger UI.
